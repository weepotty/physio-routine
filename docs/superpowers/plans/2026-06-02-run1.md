# Run 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Run 1" card to the home screen that launches a self-contained interval running session (8 × 1 min run + 2 min walk) with voice or tone audio cues.

**Architecture:** All changes are in `index.html` — a single-file app with no build step. A new fourth screen (`run-screen`) is added alongside the existing three, with its own state variables, render function, and timer logic. Audio uses the Web Speech API (voice) with a Web Audio tone fallback.

**Tech Stack:** Vanilla JS, Web Audio API (already in use), Web Speech API (`speechSynthesis`)

> **Note on TDD:** This project has no test framework. Each task includes a manual browser verification step in place of automated tests.

### Task 6: Add formatRunTime(), startRun(), and renderRun()

**Files:**
- Modify: `index.html` — JS section, after `speakPhase()`

- [ ] **Step 1: Add formatRunTime()**

After `speakPhase()`, add:
```js
function formatRunTime(seconds) {
  const m = Math.floor(seconds / 60);
  const s = seconds % 60;
  return m + ":" + (s < 10 ? "0" : "") + s;
}
```

- [ ] **Step 2: Add startRun()**

After `formatRunTime()`, add:
```js
function startRun() {
  getAudioContext();
  if (runIntervalId) { clearInterval(runIntervalId); runIntervalId = null; }
  runState = "idle";
  runPhase = "run";
  runRound = 1;
  runRemainingTime = RUN_SECONDS;
  showScreen("run");
  renderRun();
}
```

- [ ] **Step 3: Add renderRun()**

After `startRun()`, add:
```js
function renderRun() {
  const phaseLabel = document.getElementById("run-phase-label");
  const countdown = document.getElementById("run-countdown");
  const nextHint = document.getElementById("run-next-hint");
  const btn = document.getElementById("run-btn");
  const restartBtn = document.getElementById("run-restart-btn");
  document.getElementById("run-round-text").innerText = "Round " + runRound + " of " + RUN_ROUNDS;

  if (runState === "done") {
    phaseLabel.innerText = "Done!";
    phaseLabel.className = "run-phase-label run-phase-done";
    countdown.style.display = "none";
    nextHint.innerText = "";
    btn.style.display = "none";
    restartBtn.style.display = "block";
    return;
  }

  countdown.style.display = "block";
  btn.style.display = "block";
  restartBtn.style.display = "none";

  if (runPhase === "run") {
    phaseLabel.innerText = "RUN";
    phaseLabel.className = "run-phase-label run-phase-run";
    nextHint.innerText = "Walk next · " + formatRunTime(WALK_SECONDS);
  } else {
    phaseLabel.innerText = "WALK";
    phaseLabel.className = "run-phase-label run-phase-walk";
    nextHint.innerText = runRound < RUN_ROUNDS ? "Run next · " + formatRunTime(RUN_SECONDS) : "";
  }

  countdown.innerText = formatRunTime(runRemainingTime);

  const labels = { idle: "Start", running: "Pause", paused: "Resume" };
  btn.innerText = labels[runState] || "Start";
}
```

### Task 7: Add timer logic — toggleRunTimer, runTick, advanceRunPhase

**Files:**
- Modify: `index.html` — JS section, after `renderRun()`

- [ ] **Step 1: Add toggleRunTimer()**

After `renderRun()`, add:
```js
function toggleRunTimer() {
  getAudioContext();
  if (runState === "running") {
    clearInterval(runIntervalId);
    runIntervalId = null;
    runState = "paused";
    renderRun();
  } else {
    runState = "running";
    renderRun();
    runIntervalId = setInterval(runTick, 1000);
  }
}
```

- [ ] **Step 2: Add runTick()**

After `toggleRunTimer()`, add:
```js
function runTick() {
  runRemainingTime--;
  if (runRemainingTime < 0) {
    clearInterval(runIntervalId);
    runIntervalId = null;
    advanceRunPhase();
  } else {
    document.getElementById("run-countdown").innerText = formatRunTime(runRemainingTime);
  }
}
```

- [ ] **Step 3: Add advanceRunPhase()**

After `runTick()`, add:
```js
function advanceRunPhase() {
  if (runPhase === "run") {
    runPhase = "walk";
    runRemainingTime = WALK_SECONDS;
    speakPhase("walk");
    runState = "running";
    renderRun();
    runIntervalId = setInterval(runTick, 1000);
  } else if (runRound < RUN_ROUNDS) {
    runRound++;
    runPhase = "run";
    runRemainingTime = RUN_SECONDS;
    speakPhase("run");
    runState = "running";
    renderRun();
    runIntervalId = setInterval(runTick, 1000);
  } else {
    runState = "done";
    speakPhase("done");
    renderRun();
  }
}
```
